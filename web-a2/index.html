Fedora RPM packages containing A2 oberon operating system and compiler that are being developed by ETH Zürich.
Packages are available for both 64-bit & 32-bit Linuxes.
License type for A2 oberon distribution is BSD.
Packaging is being done by me (Dr. Zdenek Moravcik).
Files are available via A2 rpm repository containing packaged A2 binaries and sources.
A2-enhanced-linux64  is rpm package containing A2 distribution with some patches & enhancements added.
A2-eth-linux64  is rpm package containing original (unchanged) ETH A2 distribution
oberon-doc  is rpm package containing various A2 documentation.

To install A2 repository on your computer running Fedora OS (fedoraproject.org) execute following command:
sudo wget -O /etc/yum.repos.d/a2.repo http://83.208.14.203/repo/a2.repo After installing A2 repository install 64-bit A2 oberon by executing: sudo dnf -y install A2-enhanced-linux64 A2-eth-linux64

Optionally install 32-bit A2 oberon by executing: sudo dnf -y install A2-enhanced-linux32 A2-eth-linux32

Optionally install A2 oberon documentation by executing: sudo dnf -y install oberon-doc
