Fedora RPM packages containing A2 oberon operating system and compiler that is being developed by ETH Zurich.
Packages are available for both 64-bit & 32-bit linuxes.
License type for A2 oberon distribution is BSD.
Packaging is being done by me (Dr. Zdenek Moravcik).
Files are available via A2 rpm repository containing latest version of A2 binaries and sources.
To install this repository on your computer run following command:
wget -O /etc/yum.repos.d/a2.repo http://83.208.14.203/repo/a2.repo
a2-oberon-linux64-eth  is original ETH A2 distribution
a2-oberon-linux64-eth-patched  is A2 distribution containing some patches.

After installing A2 repository install 64-bit A2 oberon by running:
dnf install a2-oberon-linux64-eth a2-oberon-linux64-eth-patched
Optionally install 32-bit A2 oberon by running:
dnf install a2-oberon-linux32-eth a2-oberon-linux32-eth-patched
